# Predictive Analytics and Sales Forecasting

## Project Overview

This project focuses on predictive analytics using historical sales data to forecast future trends. The goal is to transform raw data into actionable insights through data preprocessing, exploratory data analysis (EDA), machine learning models, and forecasting techniques.

The project demonstrates the complete data analytics workflow, from cleaning and visualizing data to building predictive models and evaluating their performance.

---

## Objectives

- Analyze historical sales data
- Perform data cleaning and preprocessing
- Explore patterns and trends through visualization
- Build machine learning models for prediction
- Compare model performance
- Forecast future sales trends
- Generate insights for data-driven decision making

---

## Dataset

The project uses the Walmart Sales Dataset containing:

- Weekly Sales
- Fuel Prices
- Consumer Price Index (CPI)
- Unemployment Rate
- Holiday Indicators
- Store Information

Dataset Source:
https://www.kaggle.com/datasets/yasserh/walmart-dataset

## Project Workflow

### 1. Data Collection
- Downloaded historical sales dataset
- Imported data into Python environment

### 2. Data Cleaning
- Handled missing values
- Removed duplicates
- Corrected data types

### 3. Exploratory Data Analysis (EDA)
- Sales distribution analysis
- Trend visualization
- Correlation analysis
- Feature relationship analysis

### 4. Feature Engineering
- Selected important variables
- Prepared training and testing datasets

### 5. Machine Learning Models

#### Linear Regression
Used as a baseline prediction model.

#### Random Forest Regressor
Improved prediction accuracy through ensemble learning.

#### XGBoost Regressor
Applied gradient boosting techniques for enhanced performance.

### 6. Time-Series Forecasting
- Date-based trend analysis
- Future sales forecasting using ARIMA

### 7. Model Evaluation

Metrics Used:

- Mean Absolute Error (MAE)
- R² Score
- Forecast Trend Analysis
